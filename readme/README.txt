Fate Automap (Windows)
External automap tool for Fate – Gates of Dawn (Amiga)

--------------------------------------------------

Overview

Fate Automap connects to a running emulator and reads memory in real time.

Supports:
- FS-UAE
- WinUAE

--------------------------------------------------

Features

- Real-time automap
- Encounter visualization
- Player tracking
- Grid overlay
- Screenshots and memory dumps
- Debug tools

--------------------------------------------------

How to Use

1. Start emulator
2. Load Fate
3. Run FateAutomap.exe

--------------------------------------------------

Controls

Gameplay:
h Heal
m Mana
f Food
d Drink
p Piasters
t Teleport

Map:
c Turn Center Party ON/OFF
b Turn Beacon ON/OFF
g Turn Grid ON/OFF

Debug:
F3 Turn Encounter x,y ON/OFF
F4 Turn Crosshair ON/OFF
F5 Turn Hex ON/OFF
F6 Turn Tint ON/OFF
F9 Turn Inspector ON/OFF

Other:
F2 Encounter table
F7 Screenshot
F8 Memory dump

--------------------------------------------------

Output Files

Saved next to executable:
- Screenshots
- Memory dumps
- console.txt

--------------------------------------------------

Notes

- Uses Windows memory APIs
- Antivirus may flag (false positive)
- Source included for transparency

--------------------------------------------------

Troubleshooting

- Check console.txt
- Run as admin if needed
- Click window for keyboard focus

--------------------------------------------------

Enjoy exploring Fate!
